# Sport Analytics
Project about information inattention in sport markets. 

**Code structure:**

`api_calls/` - Harmonization of API calls from data providers

`database/` - Overview and construction of database

`shock_construction/` - Outline of information inattention quantification

`analysis/` - Several analyses to understand the effect of information inattentions

`automatization/` - Code overview for computing information inattention live updates and placing bets
